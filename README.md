# Mipha

Yet Another Profiler for Pythonista

written in rust

## Env Settings

### python

```
python3 -m venv venv
source ./venv/bin/activate
pip3 install -r ./requirements-dev.txt
```
### rust

```
rustup update
rustup toolchain install nightly
rustup default nightly
```

## dev

```
make test
```
