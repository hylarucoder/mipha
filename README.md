# Mipha's Grace

Yet Another Tracer for Pythonista

> 米法的祈福帮助林克从濒死状态恢复，并赋予格外的生命值

> Mipha's Grace protects Link from death and recovers his health, activating only when Link loses all of his Hearts, similarly to using a Fairy.

## 特点

- Rust, Rust, Rust
- Performance
- 无侵入代码，可直接捕获 Python 生产环境的 frame 信息
